# Sales Analysis Dashboard 📊

## 🎯 Business Problem
Analyze sales performance to identify top products, regions, and revenue trends to drive business growth.

## 📊 Dataset
- **Source**: Kaggle Superstore Dataset
- **Rows**: 9,994 sales records
- **Columns**: Order Date, Ship Date, Customer, Product, Region, Sales, Quantity, Profit

## 🛠️ Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL queries
- Power BI dashboard

## 📈 Key Analysis
1. Sales trends over time
2. Top performing regions & categories
3. Profit vs sales comparison
4. Customer purchase patterns

## 💡 Business Insights (coming soon...)

- Region Y underperforms
